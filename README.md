# IQR Home 2.0
 
IQR Home 2.0 is a structured property telemetry demo built with Next.js App Router.
 
## Requirements
 
- Node.js 20.9 or later
- npm
 
## Install
 
```bash
npm install
```
 
## Run locally
 
```bash
npm run dev
```
 
Open http://localhost:3000
 
## Production build
 
```bash
npm run build
npm run start
```
 
## Deploy to Vercel
 
- Push the project to GitHub
- Import the repository into Vercel
- Deploy with the default Next.js settings
 
## Environment variables
 
Copy `.env.example` to `.env.local` if needed.
